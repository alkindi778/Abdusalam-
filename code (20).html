نبذة عني

مهندس برمجيات وشغوف بالتصميم

كمطور Full-Stack ومصمم تطبيقات، أمتلك خبرة واسعة في بناء حلول رقمية متكاملة من الصفر. أجمع بين الخبرة التقنية في Laravel و Flutter مع فهم عميق لتجربة المستخدم والتصميم الجرافيكي، خاصة في المجال التحريري والإعلامي.

شغفي يكمن في حل المشكلات، سواء كان ذلك عبر بناء أنظمة نشر أخبار معقدة، أو تصميم واجهات برمجة تطبيقات قوية، أو أتمتة المهام باستخدام Python. أسعى دائماً لكتابة كود نظيف، قابل للتطوير، وموثوق.

مكدس التقنيات

  • Laravel & PHP
  • Flutter (Android & iOS)
  • Python (Web Scraping, Automation)
  • RESTful APIs
  • JavaScript & Tailwind CSS
  • Editorial & Layout Design

أبرز المشاريع

نظام إدارة الأخبار (CMS)

نظام متكامل مبني بـ Laravel لإدارة المحتوى الإخباري، يشمل صلاحيات المستخدمين، إدارة الوسائط، وأدوات SEO.

شاهد المشروع

موقع مكتب السياحة – عدن

موقع سياحي متجاوب ومتعدد اللغات يعرض معارض صور، نماذج استفسار، وخرائط تفاعلية.

تطوير تطبيقات الويب

بناء تطبيقات ويب متكاملة وقابلة للتطوير باستخدام Laravel و PHP، مع واجهات عصرية بـ Tailwind CSS.

تطوير تطبيقات الموبايل

تصميم وبرمجة تطبيقات موبايل (Android & iOS) عالية الأداء باستخدام Flutter.

بناء ودمج APIs

تصميم واجهات برمجة تطبيقات RESTful قوية وآمنة لربط الأنظمة والتطبيقات المختلفة.

تصميم UI/UX

تصميم واجهات مستخدم جذابة وتجارب استخدام سهلة، مع التركيز على تحقيق أهداف المشروع.

النشر الرقمي والتصميم التحريري

خبرة في تصميم تخطيطات الصحف والمجلات، وتطبيق مبادئ التصميم التحريري في المنصات الرقمية.

أتمتة وسحب البيانات

استخدام Python لأتمتة المهام المتكررة وبناء برامج لسحب البيانات من الويب (Web Scraping).
